# 如意数据大屏 (RuyBigScreen)

> 一个从 0 到 1 手把手教你制作数据可视化大屏的开源项目。

如意数据大屏（RuyBigScreen）是一个公开、免费、面向初学者的数据可视化大屏教学项目。我们希望通过一个个由浅入深的实例，帮助大家真正理解并能够**自己动手**做出一块漂亮、可用的数据大屏。

## ✨ 项目目标

- 帮助零基础同学从零开始，理解数据大屏的组成与原理
- 提供可运行的示例代码，边学边练
- 循序渐进：从静态布局 → 动态数据 → 图表组件 → 自适应适配 → 完整项目

## 📚 你将学到

- 大屏的整体布局与栅格设计
- 常用图表（折线、柱状、饼图、地图等）的绘制
- 响应式与屏幕自适应（缩放适配方案）
- 数据接入与实时更新
- 动效与视觉细节打磨

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/<your-username>/RuyBigScreen.git
cd RuyBigScreen

# 使用你喜欢的本地服务器打开（示例）
# 例如 VS Code 的 Live Server，或：
python -m http.server 8080
```
